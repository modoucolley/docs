- Click the Add Theme button to add a Theme.

### Delete

- Login as Super Admin

## Users

### View

<img
  className="block dark:hidden"
  src="/images/users-list.png"
  alt="Hero Light"
/>

- Login as a Super Admin
- Click on the "Users" icon/users module on the side-bar. 
- Users List is displayed by default

### Add

- Login as Super Admin
- Select the "Users" module on the side-bar
- Click on the Add User Button next on the Far Right of The Page
- This opens a pop up modal in which we can Add a User

<img
  className="block dark:hidden"
  src="/images/add-user.png"
  alt="Hero Light"
/>

- Fill the Form. 
- Click the Add User button to add a User.




### Edit

- Login as Super Admin
- Select the "Users" module on the side-bar.
- Click on the Edit Icon Button next on the User you want to Edit.
- This opens a pop up modal in which we can edit the chosen User.

<img
  className="block dark:hidden"
  src="/images/add-user.png"
  alt="Hero Light"
/>

- Edit the Form. 
- Click the Update User button to Edit a User.



### Delete

<img
  className="block dark:hidden"
  src="/images/users-list.png"
  alt="Hero Light"
/>

- Login as a Super Admin.
- Click on the user module on the side-bar.
- The delete icon is available on every user row.

### Block / Unblock Users

<img
  className="block dark:hidden"
  src="/images/users-list.png"
  alt="Hero Light"
/>

- Login as a Super Admin.
- Click on the user module on the side-bar.
- The Block icon in red color is available on every user row.
- Click on the Block icon
- Click "Yes" to comfirm Block or Unblock
