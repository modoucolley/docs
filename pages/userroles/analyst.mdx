### Viewing Alerts

- Navigate to the Alerts section.
- Use the filters to narrow down the alerts based on severity, date, or type.
- Click on an alert to view detailed information.

### Data Analysis Tools

- Navigate to the Data Analysis section.
- Select the dataset you want to analyze.
- Use the various tools provided to visualize and interpret the data.

### Generating Reports

- Go to the Reports section.
- Click on Generate New Report.
- Choose the parameters for your report (e.g., date range, alert types).
- Click Generate and download the report

## Themes

### Filter

<img
  className="block dark:hidden"
  src="/images/theme-search.png"
  alt="Hero Light"
/>

- Login as Analyst
- Select the Themes module on the side-bar
- Type in a keyword on the search bar on the top-left corner to search for Themes


### View


<img
  className="block dark:hidden"
  src="/images/theme-view.png"
  alt="Hero Light"
/>

- Login as a Analyst.
- Click on the Themes icon/Theme module on the left side-bar. 
- The Themes table is displayed by default.


### Add

- Login as Analyst
- Select the "Themes" module on the side-bar
- Click on the Add Theme Button next on the Far Right of The Page
- This opens a pop up modal in which we can Add a Theme

<img
  className="block dark:hidden"
  src="/images/theme-add.png"
  alt="Hero Light"
/>

- Fill the Form. 
- Click the Add Theme button to add a Theme.



### Edit

- Login as Analyst
- Select the "Themes" module on the side-bar
- Click on the edit icon next to the Theme
- Edit the Theme and click on the update button

<img
  className="block dark:hidden"
  src="/images/theme-view.png"
  alt="Hero Light"
/>


### Delete

- Login as Analyst
- Select the "Themes" module on the side-bar
- Click on the Delete icon next to the Theme
- Confirm delete by clicking on the OK button on the pop-up alert
